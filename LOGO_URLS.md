# Company Logo/Favicon URLs

Here are direct URLs you can use to download company logos:

## Easy Method: Google Favicon Service
These URLs work immediately and provide 256x256 PNG images:

1. **WhatNot**: https://www.google.com/s2/favicons?domain=whatnot.com&sz=256
2. **GEICO**: https://www.google.com/s2/favicons?domain=geico.com&sz=256
3. **Code Your Dreams**: https://www.google.com/s2/favicons?domain=codeyourdreams.org&sz=256
4. **OpenAQ**: https://www.google.com/s2/favicons?domain=openaq.org&sz=256
5. **Inclusive STEM**: https://www.google.com/s2/favicons?domain=inclusivestem.org&sz=256

**To download**: Right-click the URL above, open in new tab, then right-click the image and "Save image as..."

## Direct Website Favicons

### WhatNot
- SVG: https://www.whatnot.com/icon.svg
- PNG 32x32: https://www.whatnot.com/favicon-32x32.png
- PNG 180x180: https://www.whatnot.com/apple-touch-icon.png

### GEICO
- ICO: https://www.geico.com/favicon.ico

### Other Sites
Try these patterns (replace DOMAIN):
- https://DOMAIN/favicon.ico
- https://DOMAIN/apple-touch-icon.png
- https://DOMAIN/favicon.png

## Quick Download Script

I've created a script `get-logos.sh` that you can run:
```bash
bash get-logos.sh
```

This will download all logos to `src/assets/logos/` directory.
